7  범주형 데이터 인코딩

Keywords

python, 전처리, 통계, 기계학습, 회귀, 분류, 군집

예제 데이터셋 적재

import seaborn as sns
import pandas as pd

df = sns.load_dataset("penguins")

7.1 Label Encoding

범주를 정수 값으로 매핑하는 방식으로 순서가 없는데 숫자로 바뀌므로 주의 필요

7.1.1 예제 코드

from sklearn.preprocessing import LabelEncoder

le = LabelEncoder()

df_label = df.copy()
df_label["species_enc"] = le.fit_transform(df_label["species"])

7.3 Ordinal Encoding

순서가 있는 범주형 변수에 사용, 사용자가 순서를 직접 정의

7.3.1 예제 코드

from sklearn.preprocessing import OrdinalEncoder

oe = OrdinalEncoder(
    categories=[["small", "medium", "large"]]
)

df_ord = pd.DataFrame(
    oe.fit_transform(
        pd.DataFrame({"size": ["small", "large", "medium"]})
    ),
    columns=["size_enc"]
)

print(df_ord)
   size_enc
0       0.0
1       2.0
2       1.0

명확한 순서 존재하는 등급, 크기, 단계 표현 등에 적용

7.4 Target(Mean) Encoding

범주를 타깃 변수의 평균값으로 치환, 정보량은 크지만 데이터 누수 위험 존재

7.4.1 예제 코드

df_te = df.copy()

target = "body_mass_g"
col = "species"

mean_map = df_te.groupby(col)[target].mean()
df_te["species_enc"] = df_te[col].map(mean_map)

print(df_te)

7.5 빈도(Frequency) 인코딩

freq_map = df["island"].value_counts(normalize=True)
df_freq = df.copy()
df_freq["island_enc"] = df_freq["island"].map(freq_map)

7.6 여러 컬럼을 한 번에 인코딩

7.6.1 One-Hot (pandas)

cat_cols = ["species", "island", "sex"]

df_ohe = pd.get_dummies(df, columns=cat_cols)

7.6.2 Label Encoding (반복 처리)

from sklearn.preprocessing import LabelEncoder

df_label = df.copy()

for col in cat_cols:
    le = LabelEncoder()
    df_label[col] = le.fit_transform(df_label[col].astype(str))

7.7 인코딩 방식 요약

인코딩 순서 필요 컬럼 수 증가 추천 모델
Label X X 트리 모델
One-Hot X O 선형, 거리 기반
Ordinal O X 순서형 데이터
Target X X 고급 모델
Frequency X X 희귀 범주 많을 때
