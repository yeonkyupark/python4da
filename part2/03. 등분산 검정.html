13  등분산 검정

Keywords

python, 전처리, 통계, 기계학습, 회귀, 분류, 군집

13.1 등분산성

여러 집단의 분산이 동일하다는 가정(Homoscedasticity)

\[ \sigma_1^2 = \sigma_2^2 = \cdots = \sigma_k^2 \]

반대 개념

  • 이분산성 (Heteroscedasticity)

13.2 등분산성 검사 필요성

다음 기법들의 전제 조건으로 등분산성 검사((Test for Homogeneity of Variance) 진행

분석 등분산 필요 여부
Student t-test 필요
ANOVA 필요
선형회귀(잔차) 중요
Welch t-test 불필요
비모수 검정 불필요

정규성보다 등분산성 위반이 더 위험한 경우도 많음

13.3 등분산성 검사 시점

  • 집단 간 평균 비교 전
  • ANOVA 적용 전
  • 회귀 분석에서 잔차 진단 시

13.4 시각적 등분산성 점검

13.4.1 (1) 박스플롯

import seaborn as sns
import matplotlib.pyplot as plt

df = sns.load_dataset("penguins").dropna()
sns.boxplot(
    x="species",
    y="body_mass_g",
    data=df
)
plt.show()

관찰 포인트

  • 박스 높이
  • whisker 길이
  • 집단 간 퍼짐 차이

13.4.2 (2) 분산 비교 히스토그램

sns.histplot(
    data=df,
    x="body_mass_g",
    hue="species",
    kde=True
)
plt.show()

13.5 통계적 등분산 검정 방법

13.5.1 주요 등분산 검정 비교

검정 특징 전제
Bartlett 민감 정규성 필요
Levene 안정적 정규성 불필요
Brown–Forsythe 중앙값 기반 이상치 강건
Fligner–Killeen 비모수 매우 강건

13.6 Levene 검정

13.6.1 원리

  • 각 관측값과 집단 평균(또는 중앙값) 의 절대편차 비교 (가장 권장)

13.6.2 예제

from scipy.stats import levene

groups = [
    df[df["species"] == sp]["body_mass_g"]
    for sp in df["species"].unique()
]

stat, p_value = levene(*groups)
stat, p_value
(np.float64(5.134899089832661), np.float64(0.0063670535459093534))

판단

  • p ≥ 0.05 → 등분산 가정 가능
  • p < 0.05 → 이분산

13.7 Bartlett 검정

from scipy.stats import bartlett

stat, p_value = bartlett(*groups)
stat, p_value
(np.float64(5.692031757944016), np.float64(0.0580752393481068))

주의

  • 정규성 위반 시 잘못된 경고
  • 실제 실무에서는 거의 비권장

13.8 Fligner–Killeen 검정

from scipy.stats import fligner

stat, p_value = fligner(*groups)
stat, p_value
(np.float64(9.250672076098088), np.float64(0.009800361188170159))

특징

  • 비모수
  • 꼬리 두꺼운 분포에도 안정적

13.9 등분산성이 깨졌을 때의 선택지

13.9.1 대안 검정 사용

상황 대안
두 집단 평균 Welch t-test
다집단 평균 Welch ANOVA
전반 비모수 검정

13.9.2 분포 변환

  • 로그
  • Box-Cox
  • Yeo-Johnson

13.9.3 그룹별 분산 고려 모델

  • 가중 회귀
  • 일반화 선형 모델

13.10 정규성 ↔︎ 등분산성 관계

  • 정규성 통과 ≠ 등분산성 통과
  • Bartlett는 정규성에 의존
  • Levene/Fligner는 독립

실무 기본 조합 Q–Q plot + Levene
